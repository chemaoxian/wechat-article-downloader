# Privacy Policy / 隐私政策

**Last Updated / 最后更新**: 2026-03-10

## English Version

### 1. Data Collection

**WeChat Article Downloader does NOT collect, store, or transmit any user data.**

- All operations are performed locally on your machine
- No data is sent to external servers (except the target WeChat article URLs you request)
- No cookies, sessions, or tracking mechanisms are implemented
- No API keys, tokens, or credentials are required

### 2. Data Processing

When you use this tool to download a WeChat article:

1. **HTTP Request**: The tool sends a simple HTTP GET request to the WeChat article URL you provide
2. **Local Processing**: The HTML content is parsed and converted to Markdown/JSON locally
3. **No Persistence**: No browsing history, download logs, or user preferences are stored

### 3. User-Provided URLs

- URLs you provide are used solely for downloading the corresponding articles
- URLs are not stored, logged, or transmitted to any third party
- In verbose mode, URLs may appear in console output (visible only to you)

### 4. Third-Party Dependencies

This project uses minimal dependencies:

- **Node.js version**: Only `eslint` (development tool)
- **Python version**: Only `aiohttp` (HTTP client library)

Both dependencies are widely-used open-source packages with no known privacy concerns.

### 5. Children's Privacy

This tool is intended for developers and researchers. It is not designed for or directed to children under 13 years of age.

### 6. Changes to This Policy

We may update this Privacy Policy from time to time. Users will be notified of significant changes via the project's GitHub repository.

### 7. Contact Information

If you have any questions about this Privacy Policy, please contact us at:
- Email: 443224841@qq.com
